# Documentación de Massivo

Bienvenido. Acá vas a encontrar todo lo que necesitás para usar la plataforma:
desde tu primera campaña hasta cómo armar un bot complejo de atención al cliente
en WhatsApp.

## ¿Por dónde empezar?

- **Si es tu primera vez** → andá a [¿Qué es Massivo?](./empezar/que-es-massivo)
- **Si querés entender el modelo de la plataforma** → arrancá por [Conceptos](./conceptos/orgs-teams-usuarios)
- **Si ya estás listo para enviar** → mirá las secciones de **Email**, **WhatsApp** o **Bots** según lo que necesites

## ¿Qué cubre esta documentación?

| Sección | Para qué |
|---|---|
| **Empezar** | Tour, primer envío en 10 minutos, glosario |
| **Conceptos** | Mental model: orgs, teams, planes, multi-tenancy |
| **Email** | Dominios verificados, SMTP, templates, campañas, métricas |
| **WhatsApp** | Meta Business API, templates, inbox, conversaciones |
| **Bots** | Editor de flujo, tipos de nodos, recetas comunes |
| **Contactos** | Modelo unificado, importación, deduplicación |
| **Administración** | Usuarios, planes, audit log |
| **Referencia** | Pantalla por pantalla, atajos, códigos de error |
| **FAQ** | Las preguntas y problemas más comunes |

## ¿Faltaba algo?

Si una sección no está clara o falta cubrir algo, escribinos a
[hola@massivo.app](mailto:hola@massivo.app). Esta documentación va a crecer con
el feedback que nos den.
