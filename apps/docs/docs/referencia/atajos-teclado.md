# Atajos de teclado

Para que trabajes más rápido sin sacar las manos del teclado. **Algunos
son globales** (funcionan en toda la app), otros son **contextuales**
(solo en ciertas pantallas).

## Globales

| Atajo | Acción |
|---|---|
| `Ctrl/Cmd + K` | Abrir command palette (búsqueda rápida de cualquier pantalla) |
| `Ctrl/Cmd + /` | Abrir buscador de docs |
| `Ctrl/Cmd + B` | Toggle del sidebar (colapsar / expandir) |
| `Ctrl/Cmd + Shift + L` | Toggle dark / light mode |
| `?` | Mostrar lista de atajos disponibles en la pantalla actual |
| `Esc` | Cerrar modales, dialogs, drawers |

## Inbox de WhatsApp

| Atajo | Acción |
|---|---|
| `↑` / `↓` | Mover entre conversaciones del listado |
| `Enter` | Abrir conversación seleccionada |
| `Ctrl/Cmd + Enter` | Enviar mensaje (en el input) |
| `Shift + Enter` | Salto de línea (en el input) |
| `Ctrl/Cmd + R` | Marcar conversación actual como resuelta |
| `Ctrl/Cmd + Shift + W` | Pasar a estado "En espera" |
| `Ctrl/Cmd + T` | Tomar conversación (si está UNASSIGNED) |
| `Ctrl/Cmd + Shift + B` | Pasar al bot |
| `/` | Abrir buscador de quick replies |
| `Tab` | Expandir quick reply seleccionada |

## Tips

### Imprimir cheat-sheet

En cualquier pantalla, apretá `?` para ver los atajos disponibles
acá. Si la pantalla no soporta ningún atajo contextual, te muestra
solo los globales.

### Atajos que no funcionan

Razones comunes:

- **Browser intercepta** algunos atajos (ej. `Ctrl+T` abre nueva tab,
  no toma conversación). En esos casos, Massivo te muestra el atajo
  con un prefijo distinto o requiere foco en cierta zona.
- **Foco no está donde necesitás**: muchos atajos funcionan solo cuando
  el foco está en el área correcta. Click en la zona primero.
- **Modal abierto**: cuando hay modal/dialog activo, solo `Esc`
  funciona — para evitar cambios accidentales.

### Personalizar atajos

Hoy no se pueden customizar. Si tenés sugerencias o necesitás cambiar
alguno por conflicto (ej. con tu teclado en otro layout), escribinos
a [hola@massivo.app](mailto:hola@massivo.app).
